# Axel
Axel Accelerates All For Me !  
This is monolithic kernel written by [Rust](https://www.rust-lang.org/) and nasm.  
Support architecture is only x86 now.


# Requirements
nasm, qemu, grub, rust, cargo, ld, make

# Building
```shell
make run
```
